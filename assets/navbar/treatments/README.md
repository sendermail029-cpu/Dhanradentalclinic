Place your original navbar treatment images here.

Use these exact file names so the navbar picks them up automatically:

- `dental-implants.jpg`
- `root-canal-treatment.jpg`
- `crowns-and-bridges.jpg`
- `braces-and-aligners.jpg`
- `gum-treatments.jpg`
- `kids-dentistry.jpg`
- `dentures.jpg`
- `cosmetic-dentistry.jpg`
- `extractions.jpg`
- `oral-surgeries.jpg`
- `preventive-dentistry.jpg`
- `doorstep-dentistry.jpg`

You can use `.jpg`, `.jpeg`, `.png`, `.webp`, or `.avif`.

Then run:

```bash
npm run images:webp
```

Generated optimized files will be created in `public/navbar/treatments/` as `.webp`.

The project also runs this automatically before `npm run dev` and `npm run build`.
